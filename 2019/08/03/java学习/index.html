语言特点

通过编译成统一的字节码，不同平台下载对应JVM来实现跨平台

完全面向对象，简单，高性能，分布式，多线程，健壮

JDK（用于开发）->JRE（运行环境）->JVM(依次包含)

java SE 标准版，定位于个人计算机用户

java EE 企业版，定位于服务器端领域

java ME 微型版，定位于消费电子产品的应用

越高级语言越容易入门，但要精通还需沉下心来
